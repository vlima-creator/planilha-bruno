# 📊 Calculadora de Ponto de Equilíbrio Financeiro

Este é um aplicativo desenvolvido em **Streamlit** para ajudar empresas a calcularem seu ponto de equilíbrio financeiro. 

## 🚀 Como usar

1. Insira os **Custos Fixos Totais** da sua empresa.
2. Insira o **Preço de Venda por Unidade**.
3. Insira o **Custo Variável por Unidade**.
4. O aplicativo calculará automaticamente a margem de contribuição e o ponto de equilíbrio em unidades e receita.

## 🛠️ Instalação Local

Para rodar o projeto localmente, siga os passos abaixo:

1. Clone o repositório:
   ```bash
   git clone https://github.com/vlima-creator/planilha-bruno.git
   cd planilha-bruno
   ```

2. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```

3. Execute o aplicativo:
   ```bash
   streamlit run app.py
   ```

## 🌐 Deploy no Streamlit Cloud

Para fazer o deploy deste aplicativo:
1. Acesse [Streamlit Cloud](https://share.streamlit.io/).
2. Conecte sua conta do GitHub.
3. Selecione este repositório (`vlima-creator/planilha-bruno`).
4. Defina o arquivo principal como `app.py`.
5. Clique em **Deploy**!
